Promotional text (≤170, editable anytime)

Your 2FA codes, kept honest. Tessera generates TOTP, HOTP, and Steam Guard codes in a sleek menu-bar app — and the only Mac authenticator with a real command line.

Description

Tessera is a fast, private two-factor authenticator for macOS — and the only one with a first-class command line. Generate the 6-digit codes you already use everywhere (TOTP), plus HOTP and Steam Guard. Bring your accounts over in seconds: scan a QR code on screen, import a Google Authenticator export, or bulk-import a file of links. Reach every code from the menu bar, unlock with Touch ID, search, pin your favorites, and copy with one click. Private by design: • Your secrets are encrypted on your device with argon2id and XChaCha20-Poly1305. • No account. No servers. No tracking. Nothing leaves your Mac. • Fully open source, so anyone can verify exactly what it does. For power users, the included tess command-line tool brings a live terminal view, scripting, and JSON output — unique among Mac authenticators. Works with any service that supports standard two-factor authentication. Tessera is not affiliated with Google, Microsoft, Steam, or any other provider.

4 · App Privacy done

Choose “Data Not Collected.” Tessera keeps everything on-device, runs offline, and contains no analytics, ads, or crash SDKs. The bundled PrivacyInfo.xcprivacy already declares no tracking, no collected data, and the required-reason API (UserDefaults · CA92.1).

5 · Export compliance done

When asked “Does your app use encryption?” answer Yes, then “exempt” — Tessera uses only standard cryptography (argon2id, XChaCha20-Poly1305) for authentication/data protection, which qualifies for the exemption. The build sets ITSAppUsesNonExemptEncryption = false, so App Store Connect won’t re-ask on every upload. (Confirm this classification fits your situation; it is the standard answer for authenticator apps.)

8 · Build & upload commands

Easiest path is the Xcode GUI; CLI shown for reference.

cd tessera/swift
xcodegen generate
open Tessera.xcodeproj
# In Xcode: select the “Tessera” scheme → Signing & Capabilities → Team = Ibrahem Mahyob
# Product → Archive → (Organizer opens) → Distribute App → App Store Connect → Upload

CLI archive (replace XXXXXXXXXX with your Team ID), then upload from Xcode Organizer or Transporter:

xcodebuild -project Tessera.xcodeproj -scheme Tessera -configuration Release \
  -archivePath build/Tessera.xcarchive archive \
  DEVELOPMENT_TEAM=XXXXXXXXXX CODE_SIGN_STYLE=Automatic
